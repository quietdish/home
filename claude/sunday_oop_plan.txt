【추천 통합 프로젝트: 게임 캐릭터 시스템】
─────────────────────────────────────────

📋 요구사항:
  1. 추상 클래스 "캐릭터" 만들기
  2. 자식 클래스: "전사", "마법사", "궁수" (3개)
  3. 각 클래스는 다음을 가져야 함:
     - 속성: 이름, 체력, 공격력 (private으로 보호)
     - 메서드: 공격하기(), 정보출력() (오버라이드)

📝 구현해야 할 개념:
  1단계: 기본 클래스 구조 (속성, 메서드)
  2단계: private + getter/setter (데이터 보호)
  3단계: 상속받기 (extends)
  4단계: 메서드 오버라이드 (@Override)
  5단계: 추상 클래스 사용 (abstract)

📄 기본 구조:

abstract class 캐릭터 {
    protected String 이름;
    protected int 체력;
    private int 공격력;
    
    // 일반 메서드 (모든 캐릭터 공통)
    public void 소개하기() { ... }
    
    // 추상 메서드 (자식이 반드시 구현)
    abstract public void 공격하기();
    abstract public void 정보출력();
}

class 전사 extends 캐릭터 {
    private String 무기;
    
    @Override
    public void 공격하기() {
        System.out.println(이름 + "이(가) " + 무기 + "로 공격합니다!");
    }
    
    @Override
    public void 정보출력() { ... }
}

// 마법사, 궁수도 비슷하게 구현

💡 완성하면 이런 코드가 가능:

캐릭터[] 파티 = {
    new 전사("아서", 100, 20),
    new 마법사("메를린", 60, 35),
    new 궁수("로빈", 80, 25)
};

// for 루프 한 번이면 모든 캐릭터의 공격!
for (캐릭터 캐릭 : 파티) {
    캐릭.공격하기();
    캐릭.정보출력();
}
